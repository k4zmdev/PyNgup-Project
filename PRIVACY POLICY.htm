Ngup Privacy Policy

Introduction

This Privacy Policy outlines how Ngup, a project developed by Kazam, collects, uses, and protects the information you provide while using our software and services.

Information We Collect

Ngup does not collect any personal information or store any data provided by users unless explicitly stated. However, we may collect the following:

  • File metadata for analysis (e.g., file hash, file name, file type).
  • Details of actions performed using the tool (e.g., hash lookups, file analysis).

How We Use Your Information

The information we collect is used solely for the purpose of analyzing files and hashes, determining whether a hash is known to be malicious, and providing relevant information about the file.

Data Security

We prioritize the security of your data. Ngup does not store any sensitive information locally, and all data processing happens in real-time without saving any personal data to a server.

Cookies

Ngup does not use cookies, as it is a command-line tool and does not track or store data via a web browser.

Data Sharing

Ngup does not share any personal or identifiable information with third parties. All data processed through the tool remains solely in your local environment.

Your Rights

Since Ngup does not collect or store personal data, you do not need to request access, modification, or deletion of your data. Your use of the tool is solely for analysis and does not involve data storage or tracking by the tool.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page. We encourage you to check this page regularly for any updates.
