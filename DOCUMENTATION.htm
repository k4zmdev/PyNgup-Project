Ngup Documentation

Your Python tool to analyze file hashes and check for malicious activity.

Introduction

Ngup is a Python tool that allows you to retrieve file hashes, extract detailed file information, and check if a hash is associated with malicious activity. This tool is built to run with Python 3.12.x and provides a simple interface for users and developers.

Installation

Pre-requisites

Ensure that Python 3.12.x or later is installed on your system. You can download it from the official Python website: python.org.

Installing Ngup

  1. First, install Poetry (Python dependency manager):
    2. curl -sSL https://install.python-poetry.org | python3 -
  2. Clone the Ngup repository and install its dependencies:
    3. git clone https://github.com/your-username/ngup.git
cd ngup
poetry install

Usage

Once Ngup is installed, you can start analyzing file hashes and checking if they are malicious by using the following commands:

Basic Command

poetry run ngup-cli --hash

Example Command

To retrieve the hash of a file:

poetry run ngup-cli --hash /path/to/your/file

Check if Hash is Malicious

To check if a hash is malicious:

poetry run ngup-cli --check-malicious
